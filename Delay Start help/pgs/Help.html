Delay Start Help


Using the Delay Start window:
 
Time Remaining: The number of seconds left in the countdown.
 
Time Setting: The number of seconds that Delay Start will countdown when the program is started next time.
 
Apps to be started: A listing of apps that will be started when the countdown is finished. Add apps by pressing the + at the bottom of the list. Remove apps by selecting the apps to be removed and pressing the - at the bottom of the list.
 
Cancel: Stop the countdown.
